Parágrafos e quebras de linha

  • Você pode criar parágrafos a qualquer momento, basta colocar tudo entre o par de tags <p> e </p>

Principais formatações

  • Negrito / Destaque
    • Nessa frase temos um termo em negrito usando o par de tag <b> </b>(Não semântico)
    • Nesta frase temos um termo em destaque usando o par de tag <strong></strong>(Semântico)
  • Italico / Ênfase
    • Nesta frase tem um termo em Italico usando o par de tag <i> e </i>(Não semântico)
    • Nesta frase temos uma frase em ênfase, usando o par de tag <em> e <em>(Semântico)
  • Texto marcado
    • Podemos criar também um texto marcado usando o par de tag <mark> e </mark>
  • Texto grande | texto pequeno
    • Estamos criando um texto grande (Para aumentar letras, apenas com as CSS) e um pequeno (texto letras pequenas - par de tag small)
  • Texto deletado
    • Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado (Sêmantico - par de tag del)
  • Texto inserido
    • Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois (Sêmantico - par de tag ins)
  • Texto sobrescrito
    • Serve para inserir coisas do tipo x20+3(Usa-se o par de tag sup)
  • Texto subscrito
    • Serve para inserir coisas do tipo H2o (Usa-se o par de tag sub)

Outras formatações

  • Código fonte
    • O comando document.getElementById('teste') é escrito em linguagem Javascript(O par de tag code monoespassa o texto)
  • Citação simples
    • Como diria o pai de um amigo:O computador é um burro muito rápido(O par de tag q da sentido ao texto no qual ocorreu uma sitação)
  • Sitação completa
    • Lorem ipsum dolor sit, amet consectetur adipisicing elit. Odio nisi, molestias est a delectus veritatis ad officia velit facere quo molestiae totam odit, illum iste accusantium neque corporis! Minus, unde.
      O par de tag blockquote define uma sitação mais completa, na qual há uma quebra de linha e um deslocamento lateral, sitando um autor por exemplo.
  • Abreviações
    • Estou estudando HTML e CSS. Estou adorando!(O par de tag abbr serve para mostrar o texto abreviado das siglas)
  • Texto invertido
    • Estou aprendendo a criar coisas em HTML.(O par de tag bdo faz o texto se inverter)
